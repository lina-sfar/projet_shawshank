# Shawshank Redemption – Projet complet

## Description
Jeu développé en C avec SDL2, basé sur le film "Les Évadés" (The Shawshank Redemption).

## Fonctionnalités
- Déplacement, saut, course, attaque
- Gestion des vies et scores
- Deux joueurs (écran partagé)
- Ennemis avec IA
- Mini map
- Quiz sur le film
- Puzzle à reconstituer

## Compilation et exécution
```bash
cd integration
make clean
make
./shawshank
